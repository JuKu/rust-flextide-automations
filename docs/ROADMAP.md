# Flextide Roadmap

## Phase 1 – MVP (0–3 Months)
- Rust workspace setup
- API + Worker skeleton
- Workflow JSON model
- JS/TS sandbox execution
- WASM execution (Rust)
- Basic built-in nodes
- Next.js admin panel
- Workflow editor (React Flow)
- Local installation

## Phase 2 – Beta (3–6 Months)
- Node Marketplace (upload & install)
- Authentication & org support
- SQLx migrations
- Execution logs UI
- AI Employee management
- Templates & starter workflows

## Phase 3 – Public Launch (6–12 Months)
- Cloud hosting mode
- Role-based access control
- Team collaboration
- Analytics dashboard
- Distributed execution
- Telemetry and monitoring

## Phase 4 – Enterprise (12+ Months)
- SSO (SAML/OIDC)
- Private marketplace
- Advanced security policies
- Horizontal worker autoscaling
